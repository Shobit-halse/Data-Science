# Practical 2A - Loading Datasets

## Overview
This practical demonstrates various methods for loading and creating datasets in Python using scikit-learn and pandas.

## Topics Covered

### 1. Loading Built-in Datasets
- Loading the digits dataset from scikit-learn
- Accessing features and target values
- Understanding dataset structure

### 2. Creating Simulated Datasets

#### Regression Dataset
- Using `make_regression()` to generate synthetic regression data
- Creating datasets with specified features and targets
- Understanding feature matrices and target vectors

#### Classification Dataset
- Using `make_classification()` to generate synthetic classification data
- Creating imbalanced datasets with custom class weights
- Generating datasets with multiple features and classes

#### Clustering Dataset
- Using `make_blobs()` to generate clustered data
- Creating datasets with multiple centers
- Visualizing clusters using matplotlib scatter plots

### 3. Loading CSV Files
- Reading CSV files using pandas
- Displaying dataset information
- Working with real-world datasets (Titanic example)

## Key Libraries Used
- `sklearn.datasets` - For loading and generating datasets
- `pandas` - For CSV file handling
- `matplotlib` - For data visualization

## Learning Outcomes
- Understand different ways to load and create datasets
- Learn to work with both built-in and custom datasets
- Gain experience with synthetic data generation for testing
- Practice loading real-world data from CSV files

## Dataset Types
1. **Built-in datasets** - Pre-loaded datasets from scikit-learn
2. **Simulated datasets** - Synthetically generated data for testing
3. **CSV datasets** - Real-world data loaded from files
